# 🎥 YouTube NLP Learning Assistant

A Retrieval-Augmented Generation (RAG) application built using:

- LangChain
- FAISS
- Sentence Transformers
- Groq Llama 3
- Streamlit

The assistant answers questions from the CampusX NLP Playlist using transcript-based retrieval and LLM reasoning.

## Features

- Semantic Search using FAISS
- RAG Architecture
- Groq Llama 3 Integration
- Streamlit Deployment

## Tech Stack

Python, LangChain, FAISS, HuggingFace Embeddings, Groq, Streamlit
